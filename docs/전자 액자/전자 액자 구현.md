## 3. 구현

1. 이미지의 크기를 화면에 맞추기
2. 좌 우 스크롤 기능 구현
3. 타이머 설정하여 자동으로 이미지 넘기기
4. 이미지가 부드럽게 전환되도록, 미리 로드해두고 완벽히 제어 가능한 이미지 넘기기

## 핵심 구현
- 좌 우 넘기기 기능
- 자동 재생(넘기기) 기능
- 포인터 스와이프 기능
  - 포인터 다운 시점 기록
  - 포인터 무브 시점 기록
  - 포인터 업 시점 기록
  - 포인터 다운 ~ 업 시점까지의 이동 거리와 방향에 따라 이미지 넘기기
